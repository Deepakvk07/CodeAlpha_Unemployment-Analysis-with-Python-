## 📌 Project Overview

This project analyzes unemployment trends using Python and investigates the impact of **COVID-19** on employment and unemployment rates.

The analysis includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Regional Analysis
- COVID-19 Impact Analysis
- Data Visualization
- Policy Insights

---

## 🎯 Objectives

✅ Analyze unemployment rates.

✅ Study the impact of COVID-19.

✅ Visualize unemployment trends.

✅ Identify regional patterns.

✅ Generate policy recommendations.
